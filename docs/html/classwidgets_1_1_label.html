Detailed Description

A text label widget for displaying static or dynamic text.

The Label class provides a simple text display widget with customizable appearance including colors, styles, borders, and basic event handling.

Constructor & Destructor Documentation

◆ Label()

widgets::Label::Label ( sf::RenderWindow & parent_window,
sf::String const & string = "" )
explicit

Construct a Label with default values.

This constructor initializes a Label with the default values:

  • Position: (0, 0)
  • Text: Either string parameter or "" if parameter is omitted
  • Text Font: Default font defined in constants
  • Text Character Size: 30 pixels
  • Text color: Black
    Parameters
    parent_windowThe parent window to draw the widget to
    stringThe text string for label
